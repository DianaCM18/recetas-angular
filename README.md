# Aplicación de recetas- con Angular

## Descripción
Aplicación web desarrollada con Angular que permite consultar recetas segun la categoría seleccionada por el usuario.
La aplicación funciona como una SPA y permite navegar entre distintas vistas sin recargar la página.

## Funcionamiento
1. Página principal:
   - Explica en que consiste la aplicación.
   - Incluye un formulario donde el usuario puede seleccionar una categoría.
2. Página de recetas:
   - Muestra todas las recetas disponibles de la categoría seleccionada.
   - Cada receta incluye un botón "Ver detalle".
3. Página de detalle:
   - Muestra la información completa de la receta seleccionada.

## Tecnologías utilizadas
- Angular
- TypeScript
- HTML
- CSS
- Consumo de API experna

## Características técnicas
- Navegación entre componentes mediante Angular Router
- Comunicación entre componentes
- Servicio para obtener datos desde una API
- Aplicación estructurada siguiendo arquitectura SPA
