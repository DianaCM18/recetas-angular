# Recetas Angular App

## Descripción
Proyecto SPA (Single Page Application) desarrollado para practicar a arquitectura basada en componentes de Angular y la navegación entre rutas.
La aplicación permite visualizar y filtrar recetas por categoría.

## Estructura 
1- Página principal:
    -Se muestra una breve explicación sobre el sitio web gastronómico.
    -En la parte inferior hay un formulario que permite filtrar recetas por categoría.
2- Página de recetas por categoría:
    -Se muestran todas las recetas correspondientes a la categoría seleccionada.
    -Cada receta incluye un boton "Ver detalle"
3- Página de detalle:
    -Al pulsar "Ver detalle" el usuario accede a una nueva página.
    -Se muestran los ingredientes y la elaboración completa de la receta seleccionada.
## Tecnologías utilizadas
- Angular
- TypeScript
- HTML5
- CSS3
- Bootstrap

## Funcionalidades principales
- Consumo de API REST mediante HttpClient
- Navegación entre paginas con Angular Router
- Filtrado dinámico por categoría
- Comunicación entre componentes
- Arquitectura modular basada en componentes
